================================================================================
DATABASE SCHEMA SUMMARY
================================================================================

Tables Created:

1. Notification
   - Stores all sent notifications
   - Tracks delivery status
   - Supports retries
   - Flexible metadata (JSON)

2. NotificationPreference
   - Per-user alert settings
   - Unique per user/channel
   - 10 notification types
   - Defaults to sensible values

3. NotificationTemplate
   - Customizable per workspace
   - Supports variable interpolation
   - Per channel (email, whatsapp, etc)

Indexes:
  - workspaceId
  - userId + createdAt
  - leadId
  - channel + status
  - status + createdAt
  - workspaceId + userId
  - userId + channel
  - workspaceId + templateType

Storage estimate: ~50MB per 1M notifications
