SUMMARY
================================================================================

A zero-dependency Node.js CLI tool for automated deployment of AutomatizaWPP.

Features:
- Auto-detects deployment platform (Vercel, Docker, AWS)
- One-command deployment: npx deploy-automatizawpp
- Validates OpenGraph and SEO meta tags
- Registers with Google Search Console
- Zero external npm dependencies
- Global NPM installation support
- Colored logging with progress indicators
- Deployment reporting
